# taro-boilerplate

Taro 项目模板

## 内容组合

1. Taro
2. Taro-UI
3. 常用库

## 运行命令

```bash
1. npm install                  // 安装依赖库
2. npm run dev:weapp            // 运行小程序开发模式
```

## 目录说明

```
 - config  // taro配置
 - dist    // 发布目录
 - src     // 源码目录
    - assets      // 资源库
        - css                   // css
        - images                // 图片
    - components  // 组件库
        - biz                   // 业务组件
        - common                // 通用组件
    - pages       // 页面库
        - index
    - services    // 服务库
        - api.js                // api接口
        - config.js             // 服务器配置
        - interceptors.js       // 拦截器配置
    - utils       // 通用方法库
        - common.js             // 常见util方法合集
        - globalData.js         // 全局通用数据
        - globalUser.js         // 全局用户数据
    - app.jsx
    - app.scss
    - sitemap.json
 - .eslintrc
 - .editorconfig
 - .vmconfig                    // 项目key值列表
 - project.config.json          // 项目基本信息
```
